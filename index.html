Control Batch

Esta aplicación actúa como el sistema receptor de un lote batch. Valida el formato del archivo .txt, recalcula los totales de forma independiente y los compara contra la cabecera declarada por el sistema origen. Si todos los controles se cumplen, el lote se acepta; en caso contrario, se rechaza.
